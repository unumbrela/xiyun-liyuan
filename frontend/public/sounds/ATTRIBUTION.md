# 音效素材 · 来源与许可

本系统音效分两类：背景音乐取自公开授权录音，交互音效为本项目用 FFmpeg 合成。

## 背景音乐

- **文件**：`bgm-guqin.mp3`
- **曲目**：古琴《阳关三叠》(Guqin — Yangguan Sandie)
- **来源**：Wikimedia Commons — `File:Guqin-Yangguan_Sandie.ogg`
  https://commons.wikimedia.org/wiki/File:Guqin-Yangguan_Sandie.ogg
- **许可**：**CC BY-SA 3.0**
- **加工**：截取原曲 40s–160s 共 120 秒片段，转单声道 96kbps MP3，
  做响度归一（压缩动态、避免抢戏）与首尾淡入淡出（柔化循环接缝）。

## 交互音效（本项目合成，无版权限制）

以下短音效由 FFmpeg 用正弦波 + 指数衰减包络合成，可自由使用：

| 文件 | 用途 | 音色设计 |
|------|------|----------|
| `click.mp3` | 全局点击/按钮 | 木鱼/轻叩「笃」，中高频极快衰减 |
| `nav.mp3` | 导航切换/翻页 | 丝竹拨弦，基频 C5 + 八度 |
| `enter.mp3` | 开场入场 | 磬/钟，基频 + 非谐分音，长衰减 |
| `notify.mp3` | 提示/错误 | 轻提示两声「叮·咚」(sol→do) |

如需替换为自有/授权素材，将同名文件放入本目录即可（缺失时音效系统自动静默降级，不影响界面）。
